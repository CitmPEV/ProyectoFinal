# ProyectoFinal – Ciutat Neó

Projecte desenvolupat amb Unity (Entrega 2).

## Concepte
Ciutat futurista controlada per un sistema d’energia emocional.
Les accions del jugador modifiquen gradualment l’estat global del món
(d’entre distòpia i utopia).

## Mecàniques implementades
- Moviment del jugador
- Interacció amb objectes (tecla E)
- Sistema de World Manager amb valor continu
- Efecte local visible
- Efecte global intern amb events

## Controls
- WASD: moviment
- E: interactuar

## Estat del projecte
Entrega 2 completada.
